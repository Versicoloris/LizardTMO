🦎 LizardTMO (Too Many Outfits)

Store your transmog import codes in-game and reapply them instantly—no more external notes, no more lost outfits after prestige.

✨ What It Does

💾 Save unlimited outfit import codes
⚡ Apply outfits instantly with one click
📋 Copy codes easily when needed
🗑️ Delete and manage your outfit list
📌 Minimap button for quick access
🪶 Lightweight and simple

🕹️ Usage
Open with:
/tmo (or the minimap button)

Add a new outfit:
Enter a name
Paste your import code
Click Add

Load an outfit:
With the "View Transmog Options" window open, select a slot (ITS CONTENTS WILL BE DELETED) and click apply.

Apply → instantly loads the outfit
Copy → opens a box to copy the code
X → deletes the outfit

📦 Installation
Download the latest release
Extract into:
Interface/AddOns/
Restart the game
